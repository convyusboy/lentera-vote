Presidential Candidate API

The Presidential Candidate API is a service that provides information about candidates running for president and vice-president in the 2014 Indonesian elections. It's part of API Pemilu

Sample calls:

Candidates

List all candidates

http://api.pemiluapi.org/calonpresiden/api/caleg?apiKey=fea6f7d9ec0b31e256a673114792cb17

Get a single candidate's information

http://api.pemiluapi.org/calonpresiden/api/caleg/jk?apiKey=fea6f7d9ec0b31e256a673114792cb17

Events

Get a list of events candidates will attend, from today forward

http://api.pemiluapi.org/calonpresiden/api/events?apiKey=fea6f7d9ec0b31e256a673114792cb17

Get a list of events one candidate attended between two dates

http://api.pemiluapi.org/calonpresiden/api/events?apiKey=fea6f7d9ec0b31e256a673114792cb17&id_calon=ps&after=2014-04-20&before=2014-04-29

Promises

Get a list of promises candidates have made

http://api.pemiluapi.org/calonpresiden/api/promises?apiKey=fea6f7d9ec0b31e256a673114792cb17

Get a list of promises one candidate has made

http://api.pemiluapi.org/calonpresiden/api/promises?apiKey=fea6f7d9ec0b31e256a673114792cb17&id_calon=jw

Videos

Get a list of candidate videos

http://api.pemiluapi.org/calonpresiden/api/videos?apiKey=fea6f7d9ec0b31e256a673114792cb17

Get all videos with a certain tag

http://api.pemiluapi.org/calonpresiden/api/videos?apiKey=fea6f7d9ec0b31e256a673114792cb17&tags=debat

Candidates

Candidates related resources of the Presidential Candidate API

Candidates Collection

A list of Candidates objects with all its details

List Candidates

GET

/calonpresiden/api/caleg?apiKey={apiKey}&tahun={tahun}&jenis_kelamin={jenis_kelamin}&agama={agama}&partai={partai}&nama={nama}&limit={limit}&offset={offset}

3
Parameters
Name Description Details
offset

Number the offset from the beginning of the list of records to return (use in combination with limit to support pagination)

 number, optional
example: 300
limit

Number the number of records to return (use in combination with offset to support pagination)

 number, optional
example: 25
nama

String full or partial nama (name) of the candidate. Will return all matching records.

 string, optional
example: afzal
partai

String id of the partai (party) that the candidate is a member of

 string, optional
example: 1
agama

String full or partial agama (religion) of the candidate. Will return all matching records.

 string, optional
example: islam
jenis_kelamin

String jenis_kelamin (gender) of the candidate.

 string, optional
example: L
tahun

Number tahun (election year) that the candidate is running for.

 number, optional
example: 2014
apiKey

String apiKey of the application.

 string, required
example: 06ec082d057daa3d310b27483cc3962e

Add Comment
Parameters
Name Description Details

Number the offset from the beginning of the list of records to return (use in combination with limit to support pagination)

 number, optional

Number the number of records to return (use in combination with offset to support pagination)

 number, optional

 date, optional

 date, optional

 string, optional

 string, optional

 string, required
