Social Media Pemilu API

Social Media Pemilu API is a service that provides social media information from president candidates in the 2014 Indonesian President Elections.

Sample calls:

Post :

List all tweet
http://api.pemiluapi.org/socmedpemilu?apiKey=7b532e4991f255b768a77dbd68915939&r=post

User

List all tweet http://api.pemiluapi.org/socmedpemilu?apiKey=7b532e4991f255b768a77dbd68915939&r=user

Get a single user candidate
http://api.pemiluapi.org/socmedpemilu?apiKey=7b532e4991f255b768a77dbd68915939&r=user/view&user_id=40580714

Get list statistic of candidate account
http://api.pemiluapi.org/socmedpemilu?apiKey=7b532e4991f255b768a77dbd68915939&r=user/statistic&user_id=40580714&start_date=2014-06-01&end_date=2014-06-07

Get retweet count from particular Candidate and particular days back
http://api.pemiluapi.org/socmedpemilu?apiKey=7b532e4991f255b768a77dbd68915939&r=user/retweet&user_id=40580714&days=15

Hashtag

List all tracked Hahstag
http://api.pemiluapi.org/socmedpemilu?apiKey=7b532e4991f255b768a77dbd68915939&r=hashtag

Tag

List all tag
http://api.pemiluapi.org/socmedpemilu?apiKey=7b532e4991f255b768a77dbd68915939&r=tag

Post

Post related resources of the Post API

Post Collection

List recent Post

GET

/post&username={username}&social={social}&hashtag_id={hashtag_id}&tag_id={tag_id}&order={order}&perpage={perpage}&page={page}

Add Comment
Parameters
Name Description Details
username

String username of social media user of presidential candidate.

 string, optional
example: prabowo08
hashtag_id

Number hashtag_id is a id for hastag.

 number, optional
example: 2
tag_id

Number tag_id is a id for tag.

 number, optional
example: 1
social

String social of social media name.

 string, optional
example: twitter, facebook, instagram, or gplus
order

String order to specific if post will be sorting recent post or popular post.

 string, optional
example: recent or popular
perpage

Number perpage is a number of records to return.

 number, optional
example: 50
page

Number page is an offset from the beginning of the list of records to return

 number, optional
example: 1
