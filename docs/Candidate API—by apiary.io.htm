Candidate API

The Candidate API is a service that provides information about candidates running for office in the 2014 Indonesian elections. It's part of API Pemilu

Sample calls:

Candidates

List candidates

http://api.pemiluapi.org/candidate/api/caleg?apiKey=fea6f7d9ec0b31e256a673114792cb17&limit=25

Get a single candidate's information

http://api.pemiluapi.org/candidate/api/caleg/8200-00-0000-0029?apiKey=fea6f7d9ec0b31e256a673114792cb17

Get a list of woman candidates who were elected to the DPR

http://api.pemiluapi.org/candidate/api/caleg?apiKey=fea6f7d9ec0b31e256a673114792cb17&jenis_kelamin=p&acara_terpilih=true&lembaga=dpr

Parties

List parties

http://api.pemiluapi.org/candidate/api/partai?apiKey=fea6f7d9ec0b31e256a673114792cb17

Provinces

List provinces

http://api.pemiluapi.org/candidate/api/provinsi?apiKey=fea6f7d9ec0b31e256a673114792cb17

Electoral Districts

List electoral districts

http://api.pemiluapi.org/candidate/api/dapil?apiKey=fea6f7d9ec0b31e256a673114792cb17

Candidates

Candidates related resources of the Candidates API

Candidates Collection

A list of Candidates objects with abbreviated details (no riwayat pendidikan, pekerjaan, or organisasi)

List Candidates

GET

/candidate/api/caleg?apiKey={apiKey}&tahun={tahun}&lembaga={lembaga}&acara_terpilih={acara_terpilih}&jenis_kelamin={jenis_kelamin}&agama={agama}&provinsi={provinsi}&dapil={dapil}&partai={partai}&nama={nama}&limit={limit}&offset={offset}

Add Comment
Parameters
Name Description Details
offset

Number the offset from the beginning of the list of records to return (use in combination with limit to support pagination)

 number, optional
example: 300
limit

Number the number of records to return (use in combination with offset to support pagination)

 number, optional
example: 25
nama

String full or partial nama (name) of the candidate. Will return all matching records.

 string, optional
example: afzal
partai

String id of the partai (party) that the candidate is a member of

 string, optional
example: 1
dapil

String id of the dapil (electoral district) that the candidate is running in.

 string, optional
example: 1101-00-0000
provinsi

String id of the provinsi (province) that the candidate is running in.

 string, optional
example: 11
agama

String full or partial agama (religion) of the candidate. Will return all matching records.

 string, optional
example: islam
jenis_kelamin

String jenis_kelamin (gender) of the candidate.

 string, optional
example: L
acara_terpilih

String if true, show only candidates who were elected

 string, optional
example: true
lembaga

String lembaga (legislative body) that the candidate is running for.

 string, optional
example: DPD
tahun

Number tahun (election year) that the candidate is running for.

 number, optional
example: 2014
apiKey

String apiKey of the application.

 string, required
example: 06ec082d057daa3d310b27483cc3962e
