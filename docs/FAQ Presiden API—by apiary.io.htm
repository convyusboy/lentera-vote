FAQ Presiden API

The FAQ Presiden API is a service that provides answers to questions about rules and regulations affecting the 2014 Indonesian Presidential elections. It's part of API Pemilu

Sample calls:

List all questions

http://api.pemiluapi.org/faq-presiden/api/questions?apiKey=fea6f7d9ec0b31e256a673114792cb17

Questions

Question related resources of the FAQ Presiden API

Question Collection

A list of Question objects with abbreviated details (no riwayat pendidikan, pekerjaan, or organisasi)

List Questions

GET

/faq-presiden/api/questions?apiKey={apiKey}&tags={tags}&law={law}&limit={limit}&offset={offset}

Add Comment
Parameters
Name Description Details
offset

Number the offset from the beginning of the list of records to return (use in combination with limit to support pagination)

 number, optional
example: 300
limit

Number the number of records to return (use in combination with offset to support pagination)

 number, optional
example: 25
law

String full or partial name of a law relevant to the question. Will return all matching records.

 string, optional
example: UU No. 8 Tahun 2012
tags

String list of tags by which to filter the questions.

 string, optional
example: women,participation
apiKey

String apiKey of the application.

 string, required
example: 06ec082d057daa3d310b27483cc3962e
