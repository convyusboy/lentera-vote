Berita API

Berita API is a curated election news feed service. It's part of API Pemilu

Sample call: http://api.pemiluapi.org/berita?json=get_recent_posts&apiKey=fea6f7d9ec0b31e256a673114792cb17

List Recent Posts

Post Search

Get a list of posts matching the passed search term.

Search Posts

GET

/berita?json=get_search_results&apiKey={apiKey}&search={search}&count={count}

Add Comment
Parameters
Name Description Details
count

Integer The number of posts to return. Default is 10 if not specified.

 integer, optional
example: 2
search

String The term to search for.

 string, required
example: TPS
apiKey

String The apiKey of the application.

 string, required
example: fea6f7d9ec0b31e256a673114792cb17

Get Tagged Posts

Get a list of posts with the passed tag.

Get Tagged Posts

GET

/berita?json=get_tag_posts&apiKey={apiKey}&slug={slug}&id={id}&count={count}

Add Comment
Parameters
Name Description Details
count

Integer The number of posts to return. Default is 10 if not specified.

 integer, optional
example: 2
id

String The id of the tag. Either this or slug is required.

 string, required
example: TPS
slug

String The name of the tag. Either this or id is required.

 string, required
example: TPS
apiKey

String The apiKey of the application.

 string, required
example: fea6f7d9ec0b31e256a673114792cb17
